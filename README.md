# Rule-Based AI Chatbot

A rule-based AI chatbot built using Python Flask, HTML, CSS, and JavaScript.

## Project Overview

This project was developed as part of an Artificial Intelligence internship task. The chatbot responds to predefined user inputs using rule-based logic, input sanitization, fallback handling, and exit commands.

## Features

- Interactive chatbot interface
- Rule-based response system
- Input sanitization
- Fallback response for unknown inputs
- Exit command handling
- Flask backend API

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
